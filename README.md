# Vue TODO

> A simple TODO List

## Run the project local

**0 -** install the basic dependencies

- [NodeJS](https://nodejs.org/en/)

**1 -** Clone the project and install the dependencies:

```sh
$ git clone https://github.com/simoneas02/vue-todo
$ cd vue-todo/
$ npm install
```

**2 -** Start development mode:

```sh
$ npm run dev
$ npm run build
```
